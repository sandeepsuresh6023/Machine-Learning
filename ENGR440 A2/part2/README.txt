The program is written in python.
The program file for Part 2:Genetic Programming for Symbolic Regression is symbolicregression.py.
The program automatically takes the regression.txt file. It requires the deap machine learning library. Additionally it needs the 'numpy', 're', 'operator', 'math' and 'random' python libraries.
The program outputs the log files of the genetic evolution and the best candidate solution. The number of generations is set to 100.
